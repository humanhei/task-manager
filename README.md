## Getting Started

### Prerequisites
Please first install [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm). Then, run the following comment
```bash
npm install
```
### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
